<MidTitle>地脉</MidTitle>

| 术语 | 释义 |
|------|------|
| 世界树 | 提瓦特的所有知识与记忆都会流入世界树。其内部资料可更改。无法凭自己从世界树中删除自己的存在。<br><br>曾有一株巨木（世界树），贯彻了世界各处；这节（白银古树）枝条是它的一部分。虽然被折下，与树的联系也从来没有断开。因此，只要那棵象征着世界中心的白色巨木（白银古树）没有死去，它就永远不会枯败。<br><br>「树」是生的象征。在何处的故事中曾有一个人，在弥留之际将自己的意念托生在白杨的枝条上。如此，生命与希望就不会失落，枝条会在有阳光雨露的地方生长、抽芽，最终开花结实。<br><br>「树」同时也是智慧的象征。在何处的传说中，树上倒悬的贤人获得了书写符文操使言灵的知性，沿着树根建造的王国最终窥见了魔天的秘密。<Footnote n="1" text="初生白枝"></Footnote><br><br>一切知识、记忆与体验在大树的根茎中奔流。就如溪流汇入大河、河流汇入海洋，又化为雨水落在大地上。就如生命一般。 |
| 地脉 | 连接这个世界上一切事物的概念网络，不同的元素在其中奔流。元素力和人的精神力量在此流通，因此可以传递异国的故事。<br><br>让周边植物发光。 |

| 地灵龛 | 古代遗迹。 |
| 圣遗物 | 地脉送出的精神结晶体。 |
| 炼金术 | 地水火风，领我敕命。再造四大，重塑五行。宇宙法则相互作用，这个世界由此落成；尘世的七大元素交汇，编织了你的故事。大陆上流传的古老秘法，可以把你旅行里搜刮到的点点滴滴融合到一起。最后你会创造出世界吗？<br><br>「**黑土**」，是炼金术的词源，也是最初状态。黑土诞生白垩。创造生命的技术「**白垩**」，即是变化的开始。原初之人的质料「**赤成**」，炼金术中的意思是，情感的炼化。「**黄金**」，炼金术最后的阶段。无价值的事物终于找到了自己的意义，变化成了黄金。<br><br>摩拉本身也是一种「触媒」，是物质转化的媒介之一。也正是因此，有许多事情，必须通过摩拉才能完成。<br><br>炼金术的目标是与光明合一。 |
