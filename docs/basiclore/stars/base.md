| 术语   | 释义                                                                                                 |
| ------ | ---------------------------------------------------------------------------------------------------- |
| 星形   | 代表确定的命运。                                                                                     |
| 星空   | 宇宙是真实星空玄黑色的本质，法涅斯用蛋壳隔绝宇宙和世界的缩影，打造出虚假之天。<br>「向着星辰与深渊」 |
| 命之座 | 星空中无数的「命之座」，可能就是神之眼转化成的「最终形式」。                                         |
